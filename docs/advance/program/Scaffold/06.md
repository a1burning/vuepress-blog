## 一、关于 cli.js 报错

Q：正常情况下，我们新建完一个项目之后首先会执行 package.json 的初始化（先将 bin 以及入口js建好）。如果我们先执行了入口JS文件的设置，再去执行 npm init 以及 npm link 操作（ 有可能会出现执行环境的问题 ）

A:
结论就是
- 新建项目之后将 bin 目录里的文件新建OK ，然后执行 npm init 初始化 json 文件
- 之后再执行 npm link 是正常的
